# Simulation Analysis Project

This project contains simulation analyses, reports, figures, and supporting materials for a preregistered study on iconic duration effects in sentence interpretation.

## Quick Start

- Primary report: `reports/html/simulation_analysis_with_graphs.html`
- Alternative report: `reports/html/simulation_analysis_report.html`
- Regenerate figures: `Rscript scripts/create_visualizations.R`

## Layout

- `data/csv` - CSV outputs and summary tables
- `data/rds` - RDS outputs for scenarios and summaries
- `data/archive` - Bundled archives
- `docs` - OSF materials, preregistration exports, and review notes
- `figures` - All PNG visualizations
- `logs` - Text outputs and logs
- `reports/html` - Rendered HTML reports (with assets)
- `reports/qmd` - Quarto sources
- `scripts` - R analysis and simulation scripts
- `scripts/windows` - PowerShell helpers

## Notes

- `reports/ANALYSIS_SUMMARY.md` includes a human-friendly overview with updated paths.
- If you move files, update paths referenced in reports and scripts accordingly.
